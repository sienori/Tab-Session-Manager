Session Compressor

A tool for compressing sessions of Tab Session Manager

Load session files:

What is this

The session of Tab Session Manager stores favicons of the page. Sessions saved by TSM version 6.7.0 or lower may cause performance issues due to the large size of favicons. This tool can be used to compress favicons in sessions to reduce memory usage.
This tool works in locally. Loaded sessions will not be sent to the Internet.

How to use

  1. Export sessions in Tab Session Manager
  2. Load session files in this tool
  3. Compress!
  4. Download session files and import it into Tab Session Manager
